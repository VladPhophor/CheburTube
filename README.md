# CheburTube 🔴

> Расширение для Chrome, которое считает стоимость просмотра YouTube в рублях —  
> как будто Минцифры уже ввело плату за зарубежный трафик (150₽/ГБ, первые 15 ГБ бесплатно).

---

## Установка (локальная, без Chrome Web Store)

1. **Скачай репозиторий**
   ```
   git clone https://github.com/VladPhophor/CheburTube.git
   ```
   Или нажми **Code → Download ZIP** и распакуй.

2. **Открой Chrome** и перейди по адресу:
   ```
   chrome://extensions/
   ```

3. **Включи режим разработчика** — переключатель в правом верхнем углу.

4. Нажми **«Загрузить распакованное расширение»** и выбери папку `CheburTube` (ту, где лежит `manifest.json`).

5. Готово. Открой любое видео на YouTube — инфоборд появится в правой колонке над рекомендациями.

## Что показывает

| Блок | Описание |
|---|---|
| **Всего потрачено** | Сумма с учётом бесплатных 15 ГБ |
| **Абсолютный расход** | Сколько стоило бы всё скачанное без лимита |
| **Реалтайм** | Трафик прямо сейчас, пока смотришь |
| **Итого за ролик** | Стоимость полного просмотра в текущем качестве |
| **Ежемесячный лимит** | Прогресс использования 15 ГБ |
| **Стоимость по качеству** | Сколько стоит посмотреть это видео в каждом качестве |

### Кнопки
- **Сбросить статистику** — обнуляет накопленный счётчик трафика
- **Без лимита** — считает деньги с первого байта, игнорируя 15 ГБ
